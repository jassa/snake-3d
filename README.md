## Snake 3D

Classic Snake game, now in 3D.

## Documentation

### Controls:

- Arrow keys to move
- P to change perspective
- M key to toggle minimap (when in 3D)

### Description

There are two world views:

1. 2D (classic)
2. 3D (minimap available)
